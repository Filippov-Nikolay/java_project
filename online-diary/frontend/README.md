## Dev playground

Быстрые ручные проверки модалок и уведомлений:

- Запустить dev-сервер: `npm run dev`
- Открыть маршрут `/dev` (пример: http://localhost:3000/dev)
- Блоки на странице:
  - **Уведомления** — кнопки для каждого статуса (`successful`, `warning`, `error`, `default`) проверяют текст, цвет, анимацию появления/закрытия и автозакрытие через 5 секунд.
  - **Модалки** — кнопка открывает тестовый `CONFIRMATION` модал, чтобы проверить стек и закрытие.

Страница видима всем в dev-сборке, но предназначена только для разработчиков.
