## Dev playground

Минимальный чек для модалок, уведомлений и темы.

- Запуск dev-сервера: `npm run dev`
- Открыть: `http://localhost:3000/dev`
- Что есть на странице:
  - **Переключатель темы** (сохраняется в localStorage, по умолчанию берёт системную).
  - **Уведомления** — кнопки для статусов (`successful`, `warning`, `error`, `default`), проверка текста, цветов, анимации и автоскрытия.
  - **Модалки** — кнопка для тестового `CONFIRMATION` модала (стек/закрытие).

Переключатель темы можно использовать в любом месте: импорт `ThemeToggle` из `@features/theme/ui/ThemeToggle`. За установку темы отвечает `ThemeWatcher` (подключён в `src/app/layout.tsx`, выставляет `data-theme` на `html` и хранит выбор в `localStorage`).
