## Dev playground

Страница `/dev` для быстрых ручных проверок:

- Запуск: `npm run dev`, открыть `http://localhost:3000/dev`.
- Что есть:
  - **Переключатель темы** — учитывает системную тему при первом заходе, сохраняет выбор в `localStorage`, ранний скрипт ставит `data-theme` сразу, чтобы не мигало. Компонент доступен как `ThemeToggle`.
  - **Уведомления** — кнопки для всех статусов (`successful`, `warning`, `error`, `default`), проверка цветов, анимации, автоскрытия. Таймер 3 сек, при наведении ставится на паузу и стартует заново после ухода, можно закрыть крестиком.
  - **Модалки** — тестовый `CONFIRMATION` для проверки стека/закрытия.
  - **Новая кнопка (UI)** — градиентная, поддерживает тему, иконку слева/справа, `isLoading`, `className`.
  - **Блок NEW** — карточки, добавленные за последние 2 дня, отмечены бейджем NEW; остальные ниже по дате.

Где смотреть код:
- Витрина компонентов: `src/app/dev/helpers/showcase.tsx` (+ страница `src/app/dev/page.tsx`).
- Тема: `src/features/theme/...`, подключение в `src/app/layout.tsx`.
- Уведомления: `src/features/notifications/ui/NotificationItem`.
- Кнопка: `src/shared/ui/Button`.
